The Document Object Model (DOM) is an object-oriented representation of the web page, which can be modified with a scripting language such as JavaScript. Every web browser uses some document object model to make web pages accessible via JavaScript.The Document Object Model (DOM) is a programming interface for HTML and XML documents.

To select DOM elements, you have to use selectors such as
getElementsByTagName()
getElementsByClassName()
getElementById()
querySelector()
querySelectorAll()

Event bubbling means event propagation from inner element to outer element. You can "choose" you way of propagation with the last optional parameter of addEventListener(). In event bubbling, the event is first captured and handled by the innermost element and then propagated to outer elements.

Callback function is a function which accepts another function as an argument. and it will automatically invoke that function when it completes. A more formal definition would be - Any function that is passed as an argument to another function so that it can be executed in that other function is called as a callback function. Example! If you wanted to download data from a server (which may take long durations of time), it would be wasteful for your program or function to just freeze while you waited for the data to be fetched. Instead, it's common practice to execute the fetching operation in the background.
